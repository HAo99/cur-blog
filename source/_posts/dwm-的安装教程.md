### 安装

现将 dwm 的仓库 clone 到本地

```bash
git clone https://git.suckless.org/dwm
```

或者直接在[官网](http://dwm.suckless.org/)进行下载，下载下来的也是源码

如果想进行安装，在仓库目录下执行此命令

```bash
make clean install
```

> 若没有 make，请先进行安装
>
> ```bash
> pacman -S make
> ```
