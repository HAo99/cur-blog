### 安装

若你是 Linux 的发行版为 Arch 或 Manjaro，直接使用如下命令安装即可

```bash
pacman -S i3wm
```

> Linux 会询问你要安装哪个包，直接回车使用默认（全部）即可

若想要安装带 i3-gaps 版本的 i3wm，使用以下命令

```bash
pacman -S i3-gaps
```

> i3-gaps 是 i3wm 的 fork 版本，包含了更多特性和 i3-gaps
